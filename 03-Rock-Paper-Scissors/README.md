# 🪨 Rock Paper Scissors

A simple command-line Rock Paper Scissors game built with Python. The player competes against the computer, which makes a random choice each round.

## 🚀 Features

* Play against the computer
* Random computer choices using Python's `random` module
* Input validation for user choices
* Replay option to play multiple rounds
* Winner determination using a custom function with `return`
* Clean and beginner-friendly code structure

## 🛠️ Technologies Used

* Python 3
* `random` module

## ▶️ How to Run

1. Clone this repository.
2. Navigate to the project folder.
3. Run the program:

```bash
python rock_paper_scissors.py
```

## 🎮 How to Play

1. Enter one of the following:

   * `rock`
   * `paper`
   * `scissors`
2. The computer randomly selects its choice.
3. The game displays both choices and announces the winner.
4. Choose whether to play another round.
