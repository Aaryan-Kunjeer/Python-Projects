# Guess a Number

## Description
A command-line guessing game built with Python.

## Features
- Random number generation
- Unlimited guesses
- Too High / Too Low hints
- Attempt counter
- Input validation
- Play again option

## Technologies
- Python 3
- random module

## How to Run

```bash
python guess_a_number.py
